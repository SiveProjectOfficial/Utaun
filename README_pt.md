# 【O que é Utaun?】
Software Windows que gera bancos de voz CV para UTAU via síntese aditiva. Cria automaticamente um arquivo ZIP na sua pasta Documentos.

### 【Recursos】
* Geração automática de ZIP em Documentos.
* Criação de vogais (a, i, u, e, o, n) por síntesis aditiva.
* Compatível com consoantes externas (Ka a Pa).
* Suporta aliases em Hiragana e Romaji.
* Gera automaticamente `oto.ini`, `character.txt` e `readme.txt`.

### 【Termos】
Proibida modificação, uso comercial ou redistribuição do software. Você decide os termos dos bancos gerados.
