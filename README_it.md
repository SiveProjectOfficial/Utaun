# 【Cos'è Utaun?】
Software Windows per generare bank vocali CV per UTAU tramite sintesi additiva. Crea automaticamente un file ZIP nella cartella Documenti.

### 【Caratteristiche】
* Generazione automatica ZIP in Documenti.
* Creazione vocali (a, i, u, e, o, n) tramite sintesi additiva pura.
* Combinazione con consonanti esterne (da Ka a Pa).
* Generazione interna delle righe m / n / y / w.
* Supporto alias Hiragana e Romaji.
* Genera automaticamente `oto.ini`, `character.txt` e `readme.txt`.

### 【Termini】
Vietata la modifica, l'uso commerciale o la ridistribuzione del software. Sei libero di impostare i termini per le bank generate.
