# 【Utaun คืออะไร?】
ซอฟต์แวร์ Windows สำหรับสร้างเสียง UTAU CV ด้วยการสังเคราะห์เสียงแบบ Additive Synthesis โดยจะสร้างไฟล์ ZIP ให้ในโฟลเดอร์ Documents อัตโนมัติ

### 【คุณสมบัติ】
* สร้าง ZIP อัตโนมัติใน Documents
* สร้างเสียงสระ (a, i, u, e, o, n) ด้วย Additive Synthesis
* ผสมกับเสียงพยัญชนะภายนอกได้ (แถว Ka ถึง Pa)
* รองรับ Alias ทั้ง Hiragana และ Romaji
* สร้างไฟล์ `oto.ini`, `character.txt`, และ `readme.txt` ให้ทันที

### 【เงื่อนไขการใช้งาน】
ห้ามดัดแปลง ใช้ในเชิงพาณิชย์ หรือแจกจ่ายตัวโปรแกรมซ้ำ ส่วนเสียงที่สร้างออกมาคุณสามารถกำหนดสิทธิ์ได้เอง
