### О программе Utaun

**[Что такое Utaun?]**
Это программа для Windows, которая генерирует CV-банки для UTAU исключительно методом чисто небиологического аддитивного синтеза. Она автоматически создает ZIP-архив с аудиофайлами в папке «Документы».

**[Особенности]**
* Автоматическая генерация ZIP-банка при запуске.
* Генерация гласных (a/i/u/e/o/n) методом аддитивного синтеза.
* Генерация CV-банка в сочетании с внешними рядами согласных (от ka- до pa-).
* Внутренняя генерация рядов m/n/y/w.
* Поддержка хираганы и английских ромадзи в качестве алиасов (произношение всегда японское).
* Автоматическая генерация oto.ini / character.txt / readme.txt.
* Возможность конвертации PNG в JPG.
* readme.txt и character.txt сохраняются в кодировке Shift_JIS (ANSI) для предотвращения ошибок кодировки.
* Возможность изменения высоты голоса.
* Свободная настройка имени аудиобанка (имени персонажа).

**[Предполагаемое использование]**
* Создание UTAU-банка с небиологическим голосом.
* Создание банков для тех, кому сложно записывать собственный голос.
* Изучение материалов аддитивного синтеза голоса.
* Прототипирование самодельных аудиобанков.
* Использование в качестве материала для Human VOCALOID (распаковав ZIP).
* Использование в качестве материала для MAD/YTPMV.

**[Как использовать Utaun]**
1. Скачайте Utaun.ver-1.0.3.zip (в разделе Releases).
2. Распакуйте архив.
3. Поместите Utaun.exe в удобную папку (рекомендуется рабочий стол).
4. Запустите Utaun.exe и следуйте инструкциям на экране.
5. Когда появится сообщение «(Имя банка).zip создан в Документах!», банк готов.

**[Как использовать сгенерированный банк]**
* **UTAU**: Распакуйте ZIP, и вы получите папку, готовую к импорту в UTAU.
* **OpenUTAU**: Можно загружать ZIP напрямую (распаковка не требуется).
* ※ Информацию об использовании конкретной программы см. в её официальной справке.

**[Системные требования]**
* Windows 10 - 11 (64bit)
* **НЕ ПОДДЕРЖИВАЕТСЯ**: Звонкие гласные, шепот/дыхание, йотированные слоги (kya/kyu/kyo и т.д.), VCV (непрерывное пение), CVVC, другие специфические звуки.

**[Условия использования]**
Запрещается: модификация, редактирование, переделка, коммерческое использование, повторное распространение, декомпиляция (разбор) Utaun.exe.

**[О сгенерированном контенте]**
(Аудиофайлы, wav, icon.jpg, oto.ini и т.д.)
Условия использования контента устанавливаются вами (распространителем). Пожалуйста, пропишите желаемые правила в файле readme.txt.
