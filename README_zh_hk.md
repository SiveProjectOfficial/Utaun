# 【Utaun 係乜嘢？】
呢款係 Windows 軟件，淨係用加算合成技術生成 UTAU 專用 CV 音源。會喺你嘅「文檔」資料夾自動整好音源 ZIP 檔。

### 【特點】
* 喺「文檔」入面自動生成 ZIP。
* 用加算合成整元音（a、i、u、e、o、n）。
* 可以配搭外部子音行（Ka 行到 Pa 行）。
* 支援平假名同羅馬字 Alias。
* 自動整埋 `oto.ini`、`character.txt` 同 `readme.txt`。

### 【利用條款】
禁止改裝、商用或者再發布軟件。整出嚟嘅音源規約由你自己定。
