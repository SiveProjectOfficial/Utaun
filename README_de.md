# 【Was ist Utaun?】
Utaun ist eine Windows-Software, die UTAU-CV-Stimmbanken ausschließlich durch additive Synthese generiert. Sie erstellt automatisch eine ZIP-Datei in Ihrem Dokumente-Ordner.

### 【Features】
* Automatische ZIP-Erstellung im Dokumente-Ordner.
* Vokalerzeugung (a, i, u, e, o, n) durch reine additive Synthese.
* Kombinierbar mit externen Konsonanten (Ka- bis Pa-Reihe).
* Interne Generierung der m / n / y / w Reihen.
* Unterstützt Hiragana- und Romaji-Aliase.
* Erstellt automatisch `oto.ini`, `character.txt` und `readme.txt`.
* Konvertiert PNG zu JPG.
* Shift_JIS (ANSI) Ausgabe zur Vermeidung von Fehlzeichen.

### 【Anwendung】
1. `Utaun.ver-1.0.4.zip` von **Releases** herunterladen.
2. ZIP entpacken.
3. `Utaun.exe` starten und Anweisungen folgen.
4. Die fertige ZIP liegt in Ihrem Dokumente-Ordner.

### 【Bedingungen】
Modifikation, kommerzielle Nutzung und Weiterverbreitung der Software sind untersagt. Die generierten Stimmbanken gehören Ihnen.
