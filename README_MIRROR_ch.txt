### 关于 Utaun

**[Utaun 是什么？]**
这是一款仅通过完全非生物加法合成生成 UTAU 用 CV 音源的 Windows 软件。它会自动在文档文件夹中生成音源 ZIP 文件。

**[特点]**
* 运行时自动在文档中生成音源 ZIP。
* 通过完全非生物的加法合成生成元音（a/i/u/e/o/n）。
* 通过与外部辅音行（ka-pa 行）组合生成 CV 音源。
* 内部生成 m/n/y/w 行。
* 支持平假名别名 + 英语罗马字别名（发音均为日语）。
* 自动生成 oto.ini / character.txt / readme.txt。
* 可将 PNG 图片转换为 JPG。
* readme.txt 和 character.txt 输出为 Shift_JIS (ANSI) 以防止乱码。
* 可更改基本频率（声调）。
* 可自由设置音源名称（角色名）。

**[预期用途]**
* 想制作非生物声音的 UTAU 音源时。
* 为不擅长录制自己声音的人制作音源。
* 加法合成语音素材研究。
* 自制音源原型设计。
* 解压 ZIP 后直接用作人力 VOCALOID 素材。
* 用于音MAD、YTPMV 等素材。

**[Utaun 使用方法]**
1. 下载 Utaun.ver-1.0.3.zip（请从 Releases 下载）。
2. 解压 Utaun.ver-1.0.3.zip。
3. 将 Utaun.exe 放在任意文件夹（推荐桌面）。
4. 启动 Utaun.exe，并按照屏幕指示操作。
5. 当显示“（音源名）.zip 已生成在文档中！”时，UTAU 音源即完成。

**[生成音源的使用方法]**
* **UTAU**: 解压生成的 ZIP，即可直接作为音源文件夹读取。
* **OpenUTAU**: 可直接读取 ZIP（无需解压）。
* ※ 关于各软件的具体使用方法，请参考各自的帮助手册或官方文档。

**[运行环境]**
* Windows 10 - 11 (64bit)
* **不支持**: 浊音元音、耳语/呼吸声等表情差异、拗音（kya/kyu/kyo 等）、连续音 (VCV)、CVVC、其他特殊发音。

**[使用条款]**
禁止对 Utaun 本体 (Utaun.exe) 进行：加工、编辑、改造、商用、再分发、反编译（分解）。
