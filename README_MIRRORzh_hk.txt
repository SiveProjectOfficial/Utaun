### 關於 Utaun

**[Utaun 係咩？]**
呢款係一款淨係透過完全非生物加法合成嚟生成 UTAU 用 CV 音源嘅 Windows 軟件。佢會自動喺文件夾入面生成音源 ZIP 檔。

**[特點]**
* 執行時自動喺文件夾生成音源 ZIP。
* 透過完全非生物嘅加法合成生成元音（a/i/u/e/o/n）。
* 透過同外部輔音行（ka-pa 行）組合生成 CV 音源。
* 內部生成 m/n/y/w 行。
* 支援平假名別名 + 英語羅馬字別名（發音全部係日語）。
* 自動生成 oto.ini / character.txt / readme.txt。
* 可以將 PNG 圖片轉換做 JPG。
* readme.txt 同 character.txt 輸出為 Shift_JIS (ANSI) 以防止亂碼。
* 可以更改基本頻率（聲調）。
* 可以自由設定音源名稱（角色名）。

**[預期用途]**
* 想製作非生物聲音嘅 UTAU 音源時。
* 為唔擅長錄製自己聲音嘅人製作音源。
* 加法合成語音素材研究。
* 自製音源原型設計。
* 解壓 ZIP 後直接當做人力 VOCALOID 素材用。
* 用於音MAD、YTPMV 等素材。

**[Utaun 使用方法]**
1. 下載 Utaun.ver-1.0.3.zip（請喺 Releases 下載）。
2. 解壓 Utaun.ver-1.0.3.zip。
3. 將 Utaun.exe 擺喺任何一個文件夾（建議擺喺桌面）。
4. 啟動 Utaun.exe，並按照畫面指示操作。
5. 當顯示「（音源名）.zip 已生成在文檔中！」時，UTAU 音源就完成喇。

**[生成音源嘅使用方法]**
* **UTAU**: 解壓生成嘅 ZIP，就直接可以做音源文件夾讀取。
* **OpenUTAU**: 可以直接讀取 ZIP（唔使解壓）。
* ※ 關於各軟件嘅具體使用方法，請參考各自嘅幫助手冊或官方文檔。

**[運行環境]**
* Windows 10 - 11 (64bit)
* **唔支援**: 濁音元音、耳語/呼吸聲等表情差異、拗音（kya/kyu/kyo 等）、連續音 (VCV)、CVVC、其他特殊發音。

**[使用條款]**
禁止對 Utaun 本體 (Utaun.exe) 進行：加工、編輯、改造、商用、再分發、反編譯（分解）。
